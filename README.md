# FoodCare - Projeto Mesa Solidária

## Sobre a ONG
A FoodCare é uma organização dedicada ao combate à insegurança alimentar, conectando doadores a famílias em situação de vulnerabilidade social.

## O Projeto: Mesa Solidária
O Mesa Solidária é o sistema web desenvolvido para gerenciar o fluxo de doações da FoodCare. O software permite o controle do estoque de alimentos, monitoramento de prazos de validade e organização das entregas para as famílias cadastradas.
